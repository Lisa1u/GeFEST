# GeFEST: Geotechnical Finite Element Solve Tool
Development of a geotechnical calculation software based on the finite element method

<img width="300" height="200" alt="Logo" src="https://github.com/user-attachments/assets/e733c673-4b93-4ea4-b6bf-c6a8d0679807" />

Высокоточное современное геотехническое проектирование возможно только с опорой на численные методы расчёта. [Научно-исследовательский коллектив СПбГАСУ](https://www.researchgate.net/lab/GeFEST-Geotechnical-Finite-Element-Solve-Tool-Viacheslav-Polunin) — разрабатывает отечественный программный комплекс для моделирования оснований и сооружений методом конечных элементов.

<img width="945" height="594" alt="image" src="https://github.com/user-attachments/assets/7c607d01-7626-4367-8284-f22ebea97128" />


## Возможности

### Основные функции
- Автоматическая генерация неструктурированной сетки конечных элементов.
- Формирование расчётных моделей в плоской постановке.
- Шагово-итерационный решатель для стационарных и нестационарных задач.
- Поддержка треугольных конечных элементов высокого порядка.

### Модуль расчёта деформаций
- Математическая модель линейно-упругой среды.
- Математическая модель среды с критерием прочности Кулона–Мора.
- Модель среды со сдвиговым упрочнением и критерием прочности Мизеса–Боткина.

### Модуль расчёта температуры и фильтрации
- Математическая модель теплопроводности сплошного твердого тела.
